Diagrama de fiação para medidor de nível (Arduino, ESP8266, ESP32)

Diagrama de fiação para microcontroladores suportados e sensores de distância ultrassônicos HC-SR04 / JSN-SR04T (à prova d'água)
Revisão de 1º a 30 de julho de 2020
  • Adicionados modos ULP - Ultra low power - deep sleep para plataformas ESP8266 (Timer), ESP32 (RTC Timer).
  • O esquema foi estendido por um divisor de tensão para plataformas 3V3 ESP8266, ESP32 para sua conexão com um sensor de distância ultrassônico de 5V.
  • Adicionada a possibilidade de usar a atualização OTA em modo StandBy para plataformas ESP8266 e ESP32 em rede LAN via uploader Python.
    • Ao usar o módulo Ethernet Wiznet W5200 a W5500 com o arquivo de cabeçalho Ethernet2.h, é necessário conectar o terminal D2 Arduino ao terminal INT (INIT) no módulo Ethernet!
