Shema povezivanja mjerača razine (Arduino, ESP8266, ESP32)

Shema povezivanja podržanih mikrokontrolera i ultrazvučnih senzora udaljenosti HC-SR04 / JSN-SR04T (vodonepropusna)
Revizija 1 - 30. srpnja 2020
  • Dodani ULP modusi - Izuzetno mala snaga - duboko mirovanje za platforme ESP8266 (eksterni WAKE prekidač), ESP32 (RTC tajmer).
  • Shema proširena razdjelnikom napona za 3V3 platforme ESP8266, ESP32 za njihovo povezivanje s 5V ultrazvučnim senzorom udaljenosti.
  • Dodana mogućnost korištenja OTA ažuriranja u StandBy načinu rada za platforme ESP8266 i ESP32 u LAN mreži putem Python uploadera.
    • Kada koristite Wiznet W5200 do W5500 Ethernet modul s datotekom zaglavlja Ethernet2.h, potrebno je povezati D2 Arduino terminal s INT (INIT) terminalom na Ethernet modulu!
