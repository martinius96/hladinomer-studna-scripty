# Source codes
* compatible with ESP32, ESP8266, Rpi Zero W, Arduino (Uno / Nano / Mega)
* Arduino Core / MicroPython / ESP-IDF + FreeRTOS implementations
* WiFi, Ethernet, PHY Ethernet, LoRa, LoRaWAN, Sigfox LPWAN, NB-IoT connectivity
* Ultrasonic sensors HC-SR04T / JSN-SR04T, ToF laser sensors VL53L1X (TOF400C)
* Test web interface: https://hladinomer.eu/?lang=en
