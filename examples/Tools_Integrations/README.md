# Tools & Integrations – Examples

This folder contains **example integrations** of the Watmonitor with various popular IoT/cloud platforms and tools.

Reference Watmonitor interface: https://hladinomer.eu/?lang=en

## Available integrations

The following platforms have dedicated example code / collections in this directory:

- **ThingSpeak**  
  Classic IoT cloud platform – simple data logging, charts, MATLAB analysis, alerts

- **ThingsBoard**  
  Open-source IoT platform – dashboards, Rule Chain, custom widget, MQTT / HTTP support

- **Ubidots**  
  Modern IoT cloud service – beautiful dashboards, variables, events & notifications

- **The Things Network (TTN)**  
  LoRaWAN example payload decoder compatible with LoRaWAN MCU source codes

- **Postman**  
  Collection of ready-to-import requests for testing API for pushing / pulling data
